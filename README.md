# 🤖 Autonomous Maze-Solving Robot

A Python-powered autonomous robot that independently navigates 
mazes using Depth-First Search (DFS) algorithm, built and 
simulated in CoppeliaSim.

## 🎯 What It Does
- Navigates mazes completely independently
- Detects and avoids obstacles in real-time
- Stops when it detects a red object (color-based logic)
- Remembers visited paths to prevent revisiting routes
- Solves maze using DFS pathfinding algorithm

## 🛠️ Technologies Used
- Python
- CoppeliaSim (robotic simulation)
- Depth-First Search (DFS) algorithm
- Sensor & color detection logic

## 📁 Files In This Repository
- `maze_solver.py` — Main Python algorithm and robot logic
- `maze_scene.ttt` — Full CoppeliaSim 3D simulation scene
- Screenshots — Visual proof of robot in action
